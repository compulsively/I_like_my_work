# Local Landing Context Template Pack

## Purpose

Help each team add its own local context without hardcoding someone else's company, project, system, or workflow.

This is a reference pack, not a task executor.

## Load Order

1. Read `references/local-context-template.md`.
2. Ask the user to fill only the sections that matter for the current team.

## How To Use

1. Copy the template into the team's own workspace or knowledge base.
2. Fill stable local facts only.
3. Leave unknown or changing facts blank.
4. Keep private, sensitive, or personal information out unless the team intentionally manages it.
5. Review the template whenever product, team, or delivery rules change.

## Guardrails

- Do not invent local context.
- Do not include private credentials, secrets, or personal data.
- Do not treat the template as a legal or compliance policy.
- Keep this pack generic when sharing with other teams.
