## HTML Prototype Prompt Add-On

When asking an AI or builder to create HTML:

```text
Create a single runnable HTML prototype for [page/flow].
Use realistic mock data.
Include normal, empty, loading, error, disabled, and success states when relevant.
Keep interactions working for review: tabs, filters, modal/drawer, validation, toast, and navigation.
Do not include internal notes in the visible delivery preview.
```
