# Follow-Up Templates

## Meeting Summary

```text
Summary:
The meeting aligned on [decision/result].

This week:
1. [action] - [owner] - [deadline]
2. [action] - [owner] - [deadline]

Needs confirmation:
1. [question]
2. [question]

Risk:
[risk and proposed handling]
```

## Decision Request

```text
I recommend option [A].

Reason:
- [reason 1]
- [reason 2]

Compared with [B], [A] is better for [time/cost/risk/result].

Please confirm whether we should proceed with [A].
```

## Incomplete Information

```text
Current conclusion is not ready yet because [missing information].

Confirmed:
- [fact]

Still needed:
- [data/owner/rule]

Suggested next step:
[smallest validation action]
```

## Scope Control

```text
I suggest splitting this into three layers:

1. Current version: [must deliver now]
2. Next validation: [needs more evidence]
3. Later expansion: [not included now]

I will proceed with the current version first unless there is a different priority.
```
